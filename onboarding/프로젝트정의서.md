# 산책 관리 시스템 프로젝트 정의서

## 개요

### 프로젝트명 : *산책갈래?*

#### 목표

- 지도 API & GPS 를 이용한 사용자 거리 확인
- 지도 API & GPS 를 이용한 사용자 칼로리 확인
- 거리별 또는 칼로리 소모별 아이템 지급
- 아이템을 통한 기프티콘 or 캐릭터 꾸미기 등을 통해 앱의 만족도 상승효과 기대
- 광고, 아이템 구매를 통한 수익성 ↑
